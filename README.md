# Mod 3 Project

A King of the Hill fighting game!

TO INSTALL:

- Fork the repo
- Copy the clone link
- In your terminal run `git clone *clone-link*`

TO RUN:
Backend:

- While in the mod-3-api folder, run rails s
  Frontend:
- copy the path for index.html then go to the path in a web browser.
